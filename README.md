🕒 Tushar Clock Management System

A minimal, intelligent frontend clock system with adaptive UI, persistent preferences, and lightweight weather integration.

---

🚀 Features

- ⏱️ Real-time clock (Drift-free)
- 🔄 12h / 24h format toggle
- 🎨 Auto Light / Dark Theme (based on time)
- 💾 User preferences saved (localStorage)
- 🌦️ Live Weather (Kolkata)
- ⚡ Smooth UI animations
- 🌍 Fully responsive UI
- 🔄 Auto weather refresh

---

🧠 Tech Stack

- HTML5
- CSS3 (Glassmorphism + Animated Gradient)
- JavaScript (Vanilla JS)
- OpenWeatherMap API
